# SRS-OS
Lightweight OS for OpenComputers

## INSTALLATION

SRS-OS requires:
- A working computer
- OpenOS
- A tier 2 or higher screen (tier 1 screens dont allow touch input)
- Any gpu
- 500 KB of memory
- 30 KB of free space

1 - Install OpenOS onto your computer

2 - Reboot

3 - Run `````` in the shell

4 - Install from github

Congratulations! you installed TherOS.
